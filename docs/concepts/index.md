# Concepts

If the tutorials show the sequence of steps, the concepts pages explain why the steps work: how JSON maps to ASN.1, how trait resolution works, how the component class registry collection tools feed paccor, and how to choose the right profile family for a given deployment.

- [JSON Model](json-model.md) — the documented JSON shape paccor consumes.
- [ASN.1 Mapping](asn1-mapping.md) — how JSON values become DER-encoded ASN.1.
- [Pipeline](pipeline.md) — from .NET hardware collection to signed certificate.
- [Which Spec Version?](which-spec-version.md) — choosing v1.0, v1.1, or the v2.x family.
- [Traits](traits.md) — polymorphic Trait resolution and authoring patterns.
