# v2.1 Attribute Certificate

This flow uses the v2.x trait model with attribute certificate output.

## Inputs

Download these input files into your working directory, or substitute your own local paths:

- Issuer certificate: [TestCA.cert.example.pem][res-testca-cert]
- Issuer private key: [TestCA.private.example.pem][res-testca-key]
- Holder certificate: [TCG_EK_ecc_p384_P-384_Test.pem][res-holder-ek]
- Attributes JSON: [localhost-policyreference-v2.json][res-different-policy-v2]
- Components JSON: [componentswithtraits.json][res-test4-components-with-traits]
- Extensions JSON: [extentions.json][res-different-extensions]

If you do not want to use the linked test issuer material, use [Generate Local Demo PKI](../../getting-started.md#generate-local-demo-pki) and replace the issuer certificate and local-key filenames below.

## Verify the target Platform Certificate version

1. Reference the Attribute JSON file referenced in the [Inputs](<#inputs>) section.
2. Follow [these instructions](<../change-target-version.md#tcg-certificate-specification-definition>).
3. Verify it reflects majorVersion 2 and minorVersion 1.

## Generate the envelope

```bash
bin/paccor certgen \
  --serial 1891 \
  --not-before 20240101 \
  --not-after 20300101 \
  --issuer-cert TestCA.cert.example.pem \
  --holder-cert TCG_EK_ecc_p384_P-384_Test.pem \
  --attributes-json localhost-policyreference-v2.json \
  --components-json componentswithtraits.json \
  --extensions-json extentions.json \
  --sig-profile rsa-sha256 \
  --finalize \
  --out v21-ac-envelope.json
```

## Assemble the envelope

Choose one signing mode:

=== "Local key"

    ```bash
    bin/paccor assemble \
      --in v21-ac-envelope.json \
      --out v21-ac-cert.pem \
      --pem \
      --local-key TestCA.private.example.pem \
      --issuer-cert TestCA.cert.example.pem
    ```

=== "PKCS#11"

    ```bash
    bin/paccor assemble \
      --in v21-ac-envelope.json \
      --out v21-ac-cert.pem \
      --pem \
      --issuer-cert TestCA.cert.example.pem \
      --pkcs11-module /path/to/pkcs11.so \
      --pkcs11-slot 0 \
      --pkcs11-key-alias signing-key \
      --pkcs11-pin-file pkcs11.pin
    ```

=== "Remote signer"

    ```bash
    bin/paccor assemble \
      --in v21-ac-envelope.json \
      --out v21-ac-cert.pem \
      --pem \
      --issuer-cert TestCA.cert.example.pem \
      --remote-url https://signer.example/sign \
      --remote-auth bearer:REPLACE_ME
    ```

=== "Detached signature"

    ```bash
    bin/paccor assemble \
      --in v21-ac-envelope.json \
      --out v21-ac-cert.pem \
      --pem \
      --issuer-cert TestCA.cert.example.pem \
      --signature "$(cat v21-ac.sig.b64)" \
      --sig-encoding der
    ```

For the full option set, see [Signing Modes](../signing-options.md).

## Validate

```bash
bin/paccor validate \
  --x509v2AttrCert v21-ac-cert.pem \
  --publicKeyCert TestCA.cert.example.pem \
  --components-json componentswithtraits.json
```
