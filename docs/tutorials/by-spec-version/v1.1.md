# v1.1 Overview

> Reference: [Platform Certificate Profile v1.1 r19](https://trustedcomputinggroup.org/wp-content/uploads/IWG_Platform_Certificate_Profile_v1p1_r19_pub_fixed.pdf)

v1.1 is superseded by v2.1. Use the v1.1 family when you need legacy compatibility.

Representative ASN.1 backbone:

```asn1
PlatformConfigurationV2 ::= SEQUENCE {
  componentIdentifiers [0] IMPLICIT SEQUENCE(SIZE(1..MAX)) OF ComponentIdentifier OPTIONAL,
  componentIdentifiersUri [1] IMPLICIT URIReference OPTIONAL,
  platformProperties [2] IMPLICIT SEQUENCE(SIZE(1..MAX)) OF Properties OPTIONAL,
  platformPropertiesUri [3] IMPLICIT URIReference OPTIONAL 
```

Walkthrough:

- [v1.1 Attribute Certificate](../paccor/v1.1-attribute-certificate.md)

v1.1 supports attribute certificates only. If you need a PKC move to v2.1.
