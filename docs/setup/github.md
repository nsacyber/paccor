# GitHub Access

## Clone the repository

```bash
git clone https://github.com/nsacyber/paccor.git
cd paccor
```

## Use releases

The [GitHub Releases page](https://github.com/nsacyber/paccor/releases) is the distribution point for packaged builds.

## Use the repository-backed tutorials

The [tutorial pages](https://nsacyber.github.io/paccor/tutorials) link directly to repository-backed test inputs. Download the linked files into your working directory before running the commands or substitute with locally generated files that fit your environment.

## GitHub Actions / CI

[Paccer uses GitHub Actions](https://github.com/nsacyber/paccor/actions) to automate building and testing. They are configured in the `.github/workflows` directory and can be reviewed to learn how to build or run specific tasks.
